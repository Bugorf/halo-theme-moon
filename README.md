# Moon
在尝试了社区里几乎所有halo的主题之后，发现以下几款个人感兴趣的主题。但是每一个主题都各有优缺点，例如batterfly主题界面美观，但是过于臃肿，而stack和chirpy主题的功能和界面又过于简陋。故此决定自己开发一个新主题，取所有感兴趣的主题的优点于一体。在仔细斟酌之后，还是觉得默认主题最适合作为基础主题。因此版相对于原版在将来可能会有比较大的改动（俗称魔改），特此新开一个仓库。
